# Insynia WordPress Theme

A clean, minimalist SaaS-focused WordPress theme inspired by modern web applications like searchable.com.

## Features

- **Minimalist Design**: Clean, professional aesthetic perfect for SaaS companies and tech businesses
- **Responsive Layout**: Mobile-first design that works on all devices
- **Dual CTA Buttons**: Customizable call-to-action buttons in header (Start Free Trial, Contact Sales)
- **Modern Typography**: Clean, readable fonts optimized for web
- **SEO Optimized**: Built with SEO best practices
- **Accessibility Ready**: WCAG compliant with proper semantic markup
- **Fast Loading**: Optimized CSS and JavaScript for performance

## Customization

### Call-to-Action Buttons

Customize the header CTA buttons through **Appearance > Customize > Call to Action Buttons**:

- Primary CTA Button Text (default: "Start Free Trial")
- Primary CTA Button URL
- Secondary CTA Button Text (default: "Contact Sales") 
- Secondary CTA Button URL

### Navigation

The theme supports two navigation menus:
- **Primary Menu**: Main navigation in header
- **Footer Menu**: Links in footer area

### Widgets

The theme includes widget areas for:
- Sidebar (displayed on blog pages)
- Footer Widget 1, 2, 3 (three-column footer layout)

## Installation

1. Upload the `insynia` folder to `/wp-content/themes/`
2. Activate the theme in WordPress Admin > Appearance > Themes
3. Customize through Appearance > Customize
4. Set up menus in Appearance > Menus
5. Add widgets in Appearance > Widgets

## Screenshot

To add a theme screenshot:
1. Create a 1200x900px screenshot of your homepage
2. Save as `screenshot.png` in the theme root directory
3. The screenshot will appear in the theme selector
